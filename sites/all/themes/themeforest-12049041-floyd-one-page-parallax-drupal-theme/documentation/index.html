General Information

Thank you for choosing ST Floyd as the template for your business. This document will guide you to install and configure some basic functions of ST Floyd.

Installation

You may install in a local environment first or you can start with a hosting straight away. Drupal consumes more resources than other CMSes so do not pick a cheap hosting.

We recommend this following hosting configuration:

  • Web server: Apache 2.x on Linux hosting
  • Database: MySQL 5.0.15 or higher with PDO enabled.
  • PHP: 5.3 with PHP memory limit of 64MB.

More info on Drupal requirements is availabe at http://drupal.org/requirements

 

New Installation

New installation is recommended as it comes with all sample data so you will have a site exact as our demo. Follow the steps below to get started:

1. Download the latest version 7.x of Drupal and follow the standard installation process.

2. Please extract the floyd_fullsite_package.zip file, you will see the sites folder with 3 subfolders as:

  • sites/all/themes: contains the theme and its base theme
  • sites/all/modules: contains all modules used with this theme
  • sites/default: contains all demo images

3. Upload these three folders to your sites folder on your root directory of your website (using FTP client such as Filezilla)

4. Go back to your new website, click on Modules link, and enable the Backup and Migrate module.

 

5. Browse the Backup and Migrate tool at Configuration - System - Backup and Migrate, click on the Restore tab and upload the floyd.sql.zip found on Assets folder of the Themeforest download package. Then choose Restore now

 

6. The site is good now. Please be noticed that the admin user and password has been reset to admin/admin. You may need to change it to yours by clicking on the Hello admin link on the top right menu.

 

Done. You are good to go.

 

Install on existing websites

If you have an existing website and you want to install the theme only, please follow these steps:

  1. Install Feature module on https://drupal.org/project/features
  2. Locate the st_floyd_features.zip file under the Assets folder of the download package.
  3. Extract the zip file to /sites/all/modules.
  4. Browse Admin - Modules and enable the ST Floyd Features. Please note that you will have to install all required modules to be able to enable it.

  1. Locate the floyd_theme_package.zip file under the Assets folder of the download package.
  2. Extract the zip file to /sites/all/themes.
  3. Browse Admin - Appearance, select floyd and choose "Enable and set default"

 

All settings including views, block, context, image styles and etc ... are ready. Go to "Work with content" and "Page builder" to build your site page You can now use them as you want on your website. However, this method is recommended for experienced Drupal developers only.

Update

Frequently, there will be updates on Drupal core and contributed modules. You may need to update them to latest versions to minimize bugs and security holes.

Remember to backup your website! If there is anything wrong, you won't lose anything.

Update modules

Module updating is convenient using the admin interface. You can do it by just a few clicks.

  1. Please go to Admin – Reports – Available updates, and click to the tab Update.
  2. Check all modules you want to update, and click on the button Download these updates, a status bar will appear.
  3. After updates are downloaded successfully, please click on the Continue button.
  4. The installation process will continue and provide you a report. Scroll down to the bottom and click on Run database updates.
  5. Review your updates and click on Applying pending updates
  6. The update script will run and proceed to the result page. If everything is ok, you can now turn on your site and work as usual. All modules are up to date.

Update Drupal core

For Drupal core updating, automatic update via admin interface is not supported at the moment. You have to do it manually.

  1. Backup your existing site and database.
  2. Place the site in "off-line" mode (via Administer > Configuration > Maintenance).
  3. Download and unzip the new Drupal core files at http://drupal.org/project/drupal
  4. Upload all files and folders, except the sites folder to your hosting, overwrite existing files.
  5. Run the update.php script, just like updating modules, at http://yoursitename.com/update.php

If everything works, your Drupal site is now updated to the latest core.

Work with content

Clicking on the Add content button on the top left, you will proceed to the next screen where you can choose content type.

 

The content type names speak for itself:

  • Backstretch is background slideshow image. To configure backstretch please select "Backstretch" as reaction in your context (example: "admin/structure/context/list/home/edit").
  • Portfolio is a project showcase, displaying on the Portfolio page.
  • Member is a team member
  • Slide is a content slider.
  • ...

Just click on the content type you want to create, fill in information and click on Save.

Menu

To add, edit or delete any menu items, you need to access to the Menu configuration page by click on Structure on the top admin menu, then select Menu. It will display the list of menus.

There are 5 menus, but we concern the Main menu most. Please click on list links to see the list of menu items.

 

Here, you can click on Add link to add a new menu item, Edit to edit an existing item. You can also drag and drop these items to change their displaying orders.

When add or edit a menu item, please fill in the following info:

  • Menu Link Title: the menu item title
  • Path: you can enter the absolute path (for example, node/4) or relative path (for example, about-us).
  • Description: the text shown when hovering to the menu item
  • Enabled: select it to activate the menu item, otherwise it won't show up.
  • Show as expanded: if selected and the menu has children, the menu will always appear expanded.
  • Parent link: the parent menu item.
  • Weight: the order of the item, the least will show up first.
